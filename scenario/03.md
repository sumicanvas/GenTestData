## 시나리오 3번 : 종목코드+검색어 있는 경우

  - 뉴스구분 X
  - 종목코드 O
  - 검색어  O
  - 검색어가 DB에 있는지 없는지는 결과 건수로 판단
  - 전체 데이터 대상, 최신 뉴스 우선

```js
db.news_array.aggregate([
  {
    $search: {
      index: "news_search_index",
      compound: {
        must: [
          {
            text: {
              query: "현대차",
              path: ["title", "contents"],
              matchCriteria: "all"
            }
          }
        ],
        filter: [
          {
            equals: {
              path: "shcode",
              value: "005380"
            }
          }
        ]
      },
      sort: {
        score: { $meta: "searchScore" },
        newscode_ts: -1
      }
    }
  },
  { $limit: 100 },
  {
    $project: {
      _id: 1,
      newscode_ts: 1,
      title: 1,
      contents: 1,
      dgubun: 1,
      shcode: 1,
      kind: 1,
      score: { $meta: "searchScore" }
    }
  }
]);
```
