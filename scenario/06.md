## 시나리오 6번 : 특정뉴스원에 대한 검색( 종목코드 포함)

  - 종목코드 O
  - 뉴스구분 O
  - 검색어 X
  - 해당 종목 + 해당 뉴스매체의 전체 뉴스 최신순 조회
    
```js
db.news_array.aggregate([
  {
    $search: {
      index: "news_search_index",
      compound: {
        filter: [
          {
            equals: {
              path: "shcode",
              value: "000660"
            }
          },
          {
            equals: {
              path: "dgubun",
              value: "4"
            }
          }
        ]
      },
      sort: {
        newscode_ts: -1
      }
    }
  },
  { $limit: 10 },
  {
    $project: {
      _id: 1,
      newscode_ts: 1,
      title: 1,
      contents: 1,
      dgubun: 1,
      shcode: 1,
      kind: 1
    }
  }
]);
```
