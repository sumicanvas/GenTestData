## 시나리오 5번 : 종목코드만 있을 경우 (전체뉴스)

  - 종목코드 O
  - 뉴스구분 X
  - 검색어 X
  - 해당 종목의 전체 뉴스 최신순 조회
```js
//요렇게 해봤지만 결과는 다르지 않음 그냥 종목코드 000660 이 들어가 있는 것을 모두 찾아서 시간순으로 정렬
      sort: {
        score: { $meta: "searchScore" },
        newscode_ts: -1
      }
```
```js
db.news_array.aggregate([
  {
    $search: {
      index: "news_search_index",
      compound: {
        filter: [
          {
            equals: {
              path: "shcode",
              value: "000660"
            }
          }
        ]
      },
      sort: {
        newscode_ts: -1
      }
    }
  },
  { $limit: 100 },
  {
    $project: {
      _id: 1,
      newscode_ts: 1,
      title: 1,
      contents: 1,
      dgubun: 1,
      shcode: 1,
      kind: 1
    }
  }
]);

```
