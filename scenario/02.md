## 시나리오 2번 : 검색어 + 종목코드 filter

```js
db.news_array.aggregate([
|   {
|     $search: {
|       index: "news_search_index",
|       compound: {
|         filter: [
|           {
|             equals: {
|               path: "shcode",
|               value: "005930"
|             }
|           }
|         ]
|       },
|       sort: {
|         newscode_ts: -1
|       }
|     }
|   },
|   { $limit: 5 },
|   {
|     $project: {
|       _id: 1,
|       newscode_ts: 1,
|       title: 1,
|       dgubun: 1,
|       shcode: 1,
|       kind: 1,
|       score: { $meta: "searchScore" }
|     }
|   }
| ])
```
