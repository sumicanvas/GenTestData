## 시나리오 2번 : 특정뉴스원에 대한 뉴스제목 전체 검색
  시나리오 2
  - 뉴스구분 O
  - 종목코드 X
  - 검색어 X
  - 해당 뉴스매체 전체 뉴스 최신순 조회

```js
db.news_array.aggregate([
  {
    $search: {
      index: "news_search_index",
      compound: {
        filter: [
          {
            equals: {
              path: "dgubun",
              value: "P"
            }
          }
        ]
      },
      sort: {
        newscode_ts: -1
      }
    }
  },
  { $limit: 5 },
  {
    $project: {
      _id: 1,
      newscode_ts: 1,
      title: 1,
      contents: 1,
      dgubun: 1,
      shcode: 1,
      kind: 1
    }
  }
])
```
