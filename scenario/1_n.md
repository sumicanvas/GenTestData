## 시나리오 1 : 종목코드와 검색어 모두 없는경우- 뉴스제목 전체
  - 종목코드 X
  - 뉴스구분 X
  - 검색어 X
  - 전체 뉴스 최신순 조회

```js
db.news_array.aggregate([
|   {
|     $search: {
|       index: "news_search_index",
|       compound: {
|         filter: [
|           {
|             exists: {
|               path: "newscode_ts"
|             }
|           }
|         ]
|       },
|       sort: {
|         newscode_ts: -1
|       }
|     }
|   },
|   { $limit: 10 },
|   {
|     $project: {
|       _id: 1,
|       newscode_ts: 1,
|       title: 1, contents: 1,
|       dgubun: 1,
|       shcode: 1,
|       kind: 1
|     }
|   }
| ])


```
