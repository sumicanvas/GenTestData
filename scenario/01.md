## 시나리오 1 : 종목코드와 검색어 모두 없는경우- 뉴스제목 전체
  - 종목코드 X
  - 뉴스구분 X
  - 검색어 X
  - 전체 뉴스 최신순 조회

- 이 경우에는 검색어는 있지만 뉴스 제목 + 내용에 해당 내용이 없는 경우
- ***결과는 아무것도 안나옴***
```js
db.news_array.aggregate([
  {
    $search: {
      index: "news_search_index",
      text: {
        query: "에헤라",
        path: ["title", "contents"]
      },
      sort: {
        newscode_ts: -1
      }
    }
  },
  { $limit: 5 },
  {
    $project: {
      _id: 1,
      newscode_ts: 1,
      title: 1,
      dgubun: 1,
      shcode: 1,
      kind: 1
    }
  }
])

```

```js
db.news_array.aggregate([
  {
    $search: {
      index: "news_search_index",
      compound: {
        filter: [
          {
            exists: {
              path: "newscode_ts"
            }
          }
        ]
      },
      sort: {
        newscode_ts: -1
      }
    }
  },
  { $limit: 10 },
  {
    $project: {
      _id: 1,
      newscode_ts: 1,
      title: 1,
      contents: 1,
      dgubun: 1,
      shcode: 1,
      kind: 1
    }
  }
])

```
